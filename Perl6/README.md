# What
* 在 2015 年十二月發布了第一個正式版 (v6.c)
* 內建一套完整的物件系統
  * 封裝、繼承、Role (類似 Java 8 的介面)、函式重載、運算子重載、多型
* Grammer，將語法分析器 (parser) 內建在語言特性中
  * 解析 HTML、XML 或其他複雜的文字檔案
* 修改 Perl 5 的一些歷史議題
  * 和 Perl 5 在語法上有相當的差異，可視為一個全新的語言
